# myaircon.online

online air conditioner https://myaircon.online

- 로컬로 실행하려면 터미널에 npm install & node server.js
- Web Audio API의 audioContext.audioWorklet.addModule()은 로컬호스트 또는 https 환경에서만 실행가능
